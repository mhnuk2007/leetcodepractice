# DSA Repository Structure (Java)

This document explains the folder organization used in this repository.
The goal is to keep solutions clean, organized, and easy to navigate.

## Repository Layout
```
dsa-java
  .idea
  out
    production
      LeetCodePractice
  README.md
  DSA_NOTES.md
  LEETCODE_SOLUTIONS_INDEX.md
  DSA_REPOSITORY_STRUCTURE.md
  DSA_TRACKER.md

  src
    arrays
    arrays101
    dailychallenges
    dp
    strings
    linkedlist
    patterns
    slidingwindow
    sortalgrithms
    twopointer
```

## Generated and IDE Folders
- `.idea`: IDE configuration (IntelliJ). Safe to ignore for code review.
- `out`: compiled class output (generated by the IDE or build tooling).

## Tips for Maintaining This Repo
- Write clean code
- Add problem number in comments
- Add time complexity
- Add space complexity
- Write a short explanation

Example:
```java
/*
LeetCode 206
Reverse Linked List

Time Complexity: O(n)
Space Complexity: O(1)
*/
```
