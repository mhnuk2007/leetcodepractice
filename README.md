## 🎯 PROBLEM-SOLVING APPROACH

### Step-by-Step Method:

1. **Understand the Problem**
   - Read carefully, identify inputs/outputs
   - Ask clarifying questions
   - Work through examples manually

2. **Plan Your Approach**
   - Identify the pattern (two pointers, sliding window, etc.)
   - Think about edge cases
   - Consider time/space complexity

3. **Implement in Java**
   - Write clean, readable code
   - Use meaningful variable names
   - Add comments for complex logic

4. **Test Thoroughly**
   - Test with provided examples
   - Test edge cases (empty, single element, large input)
   - Test boundary conditions

5. **Optimize**
   - Can you reduce time complexity?
   - Can you reduce space complexity?
   - Is there a better algorithm?

6. **Review**
   - Look at other solutions
   - Understand different approaches
   - Note patterns for similar problems

## 💻 JAVA-SPECIFIC TIPS

### Essential Data Structures in Java:

```java
// Arrays
int[] arr = new int[10];
Arrays.sort(arr);
Arrays.fill(arr, 0);

// ArrayList
List<Integer> list = new ArrayList<>();
Collections.sort(list);
Collections.reverse(list);

// LinkedList
LinkedList<Integer> linkedList = new LinkedList<>();

// Stack
Stack<Integer> stack = new Stack<>();
// Or use Deque: Deque<Integer> stack = new ArrayDeque<>();

// Queue
Queue<Integer> queue = new LinkedList<>();

// PriorityQueue (Min Heap by default)
PriorityQueue<Integer> minHeap = new PriorityQueue<>();
PriorityQueue<Integer> maxHeap = new PriorityQueue<>(Collections.reverseOrder());

// HashMap
Map<Integer, Integer> map = new HashMap<>();
map.put(1, 1);
map.getOrDefault(1, 0);

// HashSet
Set<Integer> set = new HashSet<>();

// TreeMap (sorted)
TreeMap<Integer, Integer> treeMap = new TreeMap<>();

// TreeSet (sorted)
TreeSet<Integer> treeSet = new TreeSet<>();
```

### Common Java Patterns:

```java
// Custom Comparator
PriorityQueue<int[]> pq = new PriorityQueue<>((a, b) -> a[0] - b[0]);

// Sorting with lambda
Arrays.sort(intervals, (a, b) -> a[0] - b[0]);

// Stream operations
int sum = list.stream().mapToInt(Integer::intValue).sum();

// StringBuilder for string manipulation
StringBuilder sb = new StringBuilder();
sb.append("text");
String result = sb.toString();
```

## 📊 COMPLEXITY ANALYSIS GUIDE

### Time Complexity Cheat Sheet:

- **O(1)**: Hash table operations, array access
- **O(log n)**: Binary search, balanced tree operations
- **O(n)**: Single pass through array, linear search
- **O(n log n)**: Efficient sorting (merge sort, heap sort)
- **O(n²)**: Nested loops, bubble sort
- **O(2ⁿ)**: Recursive solutions with branching
- **O(n!)**: Permutations

### Space Complexity:

- Consider: recursion call stack, auxiliary data structures
- In-place algorithms: O(1) extra space
- Recursive DFS: O(h) where h is height of tree
